Data Management

Supabase Setup Required

To enable full cloud storage, create these in your Supabase dashboard:

1. Create Storage Bucket

  1. Go to Supabase Dashboard
  2. Select your project → Storage
  3. Click New Bucket
  4. Name it: nexus-files
  5. Check Public bucket
  6. Click Create bucket

2. Create Database Table

Go to SQL Editor and run:

CREATE TABLE IF NOT EXISTS files (
    id TEXT PRIMARY KEY,
    name TEXT NOT NULL,
    type TEXT,
    size BIGINT,
    folder TEXT,
    cloud_url TEXT,
    cloud_path TEXT,
    created TIMESTAMPTZ DEFAULT NOW(),
    modified TIMESTAMPTZ DEFAULT NOW(),
    starred BOOLEAN DEFAULT FALSE,
    synced BOOLEAN DEFAULT TRUE
);

CREATE TABLE IF NOT EXISTS sql_tables (
    name TEXT PRIMARY KEY,
    columns JSONB,
    rows JSONB,
    created TIMESTAMPTZ DEFAULT NOW()
);

3. Set RLS Policies (Optional)

-- Allow public read/write for demo
ALTER TABLE files ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Allow all" ON files FOR ALL USING (true);

-- Storage policies
CREATE POLICY "Allow public uploads" ON storage.objects 
FOR INSERT WITH CHECK (bucket_id = 'nexus-files');

CREATE POLICY "Allow public downloads" ON storage.objects 
FOR SELECT USING (bucket_id = 'nexus-files');
